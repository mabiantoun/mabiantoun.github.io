Class Overview

This class is about software analysis tools and will cover the following topics:

  • Static analysis tools:
    • Principles of static analysis
    • Applications
    • Assignments: write a plugin using the Eclipse Abstract Syntax Tree to perform some static analysis
  • Dynamic analysis tools:
    • General principles for dynamic analysis and infrastructure
    • Applications
    • Assignments: run a dynamic analysis tool
  • Evaluation of tools:
    • Exploratory and evaluative studies
    • Analytical studies (case studies, extended examples)
    • Assignment: evaluate research/commercial tool
  • Other assignments::
    • Probe a software system for security violations
    • Analyze the code structure of a system

Prerequisites

  • Working knowledge of the Java programming language or similar (C++, C#, etc.)
  • Good knowledge of software engineering principles (CSC 4110, CSC 6110, or equivalent)
  • Or instructor's approval

Evaluation

Evaluation will be based on the following approximate percentages:
  • Assignments (~60%)
    • Demonstrate a basic understanding of analysis techniques/li>
    • Evaluate engineering tradeoffs
  • Course Project (~15%)
    • Evaluate analysis tool
    • Written report and in-class presentations
    • Write and apply custom analyses
  • Written Exam (Mid-Term) (~15%)
    • Theory and engineering
  • Class Participation and Readings (~10%)
    • Reading questions on papers from the literature
    • Discussion, presentations in class

Late Work Policy

Each student has 5 late days for the entire semester which can be used in any assignment, so long as the assignment hand-in is not on the critical path for the following assignment.  When turning in your work, just let the instructor know how many late days you are using for that assignment. Work turned in late after the late days are used up will only be accepted under extraordinary circumstances, such as a documented medical emergency. In other words, after you use all your late days, you will get a grade of zero for every assignment that you turn in late.
